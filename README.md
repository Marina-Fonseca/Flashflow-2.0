# FlashFlow - Flashcards Interativos

FlashFlow é uma aplicação web para estudar usando flashcards, com interface moderna, responsiva e personalizável. Desenvolvido com HTML, CSS e JavaScript no VS Code.

## Funcionalidades

- Exibe perguntas e respostas de flashcards.
- Efeito de virar o card para mostrar a resposta.
- Contador de cards.
- Botões para avançar para o próximo card e voltar para o início.
- Suporte a múltiplas categorias de cards (ex: Programação, Inglês) com seletor no topo.
- Adição de novos cards diretamente pela interface.
- Edição e exclusão de cards salvos.
- Cards e categorias salvos no navegador (localStorage).
- Layout responsivo e visual moderno.

## Estrutura de Arquivos

```
c:\Dev\flashflow\
│
├── index.html      # Página principal
├── styles.css      # Estilos personalizados
├── scripts.js      # Lógica dos flashcards
├── README.md       # Descritivo do Projeto
```

## Como usar

1. Abra o arquivo `index.html` em seu navegador.
2. Escolha a categoria desejada no topo da página.
3. Navegue pelos flashcards usando os botões "Próximo Card" e "Voltar para o início".
4. Clique no card para ver a resposta.
5. Adicione, edite ou exclua cards usando os botões e formulários disponíveis.

## Tecnologias

- HTML
- CSS
- JavaScript

## Personalização

- Para adicionar mais flashcards por padrão, edite o objeto `defaultData` no arquivo `scripts.js`.
- Para alterar estilos, modifique o arquivo `styles.css`.
